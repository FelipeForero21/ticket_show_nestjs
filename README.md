```markdown
# Pasos para empezar el proyecto

## Instalación de dependencias
Abre una terminal y ejecuta el siguiente comando para instalar las dependencias del proyecto: `npm install`.

## Levantar la base de datos
En otra terminal, asegúrate de tener Docker instalado y ejecuta el siguiente comando para levantar la base de datos: `docker-compose up`.

## Correr el proyecto
Una vez que las dependencias estén instaladas y la base de datos esté corriendo, en la primera terminal ejecuta el siguiente comando para iniciar el proyecto en modo desarrollo: `npm run start:dev`.

## Configuración de variables de entorno
Renombra el archivo `.env.example` a `.env` y asegúrate de configurar las variables de entorno necesarias, incluyendo las claves de acceso y las variables de la base de datos.
```
